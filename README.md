## <ins>Usage</ins>

1. Open a terminal or command line.

2. Install the requirements.
    ```bash
    pip3 install -r requirements.txt
    ```

3. Navigate to the folder bioBuddy.
    ```bash
    cd user/Desktop/bioBuddy
    ```
4. Run the main.py file with python3 interpreter.
    ```bash
    python3 main.py
     ```
